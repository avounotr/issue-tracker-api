## Setup the application locally  

- Download API repo
- Create .env file. It should looks like:  

```
CLIENT_ID='799471d8f0e34096e320'
CLIENT_SECRET='b80c24ecbdb44c################'
```  

## Run the API locally

- npm install
- npm start
